# Piano Screenshot per Manuale Utente Namo

Questo documento elenca tutti i 39 screenshot necessari per il manuale utente.
Per ogni screenshot vengono fornite le versioni **desktop** (1280x800) e **mobile** (375x812).

## Convenzione di naming

- Desktop: `{label}.png`
- Mobile: `{label}-mobile.png`

## Note

- Tutti gli screenshot vengono catturati con il server di sviluppo in esecuzione su `localhost:3000`
- Il viewport desktop è 1280x800, il mobile è 375x812
- Se uno screenshot non può essere catturato per mancanza di dati, viene documentato in `MISSING_DATA.md`
- Lo script di automazione è in `take-screenshots.mjs`
